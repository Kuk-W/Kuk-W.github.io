# 红黑树

定义

  1. 每个节点是红的或者黑的
  2. 根节点是黑的
  3. 每个叶子节点都是黑的
  4. 如果一个节点是红的，则他的两个儿子都是黑的
  5. 对于每个节点，从该节点到其子孙节点的所有路径上都包含相同数目的黑节点
# 红黑树的插入

插入的节点 是红色的定义 方便

总结

  1. 在插入时 先确定插入的位置 如果插入的位置的父节点是黑色 不进行调整
  2. 父节点是红色
    1. 叔叔 是空的， 需要旋转 + 变色
    2. 叔叔 红色的， 需要把 叔叔和爸爸 变成黑色 + 把爷爷变成红色 （如果爷爷节点是根节点 则需要变回黑色）
    3. 叔叔 黑色的， 旋转 + 变色（思考）
  3. 优先考虑 祖孙三代

其实只保证黑色的树保证平衡
